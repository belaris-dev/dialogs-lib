# Belaris Dialogs

The easy way to create AlertDialogs in Android projects.

## How to

### Step 1

Add it in your root build.gradle:

```
allprojects {
  repositories {
    ...
    maven { url 'https://jitpack.io' }
  }
}
```

### Step 2

Add the dependecy:

```
dependencies {
  compile 'com.github.belaris-dev:dialogs-lib:1.0.0'
}
```
